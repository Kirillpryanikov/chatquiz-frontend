# Quick guide

```git clone https://github.com/Kirillpryanikov/chatquiz-frontend```

## For run quiz application:
```cd quiz```
```npm install```
```npm install -g gulp``` (if needed)
```gulp```
```ionic serve```


## For run chat application:
```cd imagechat```
```npm install```
```npm install -g gulp``` (if needed)
```gulp```
```ionic serve```
